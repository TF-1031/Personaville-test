# Personaville Roadmap

## v0.1 — Core Foundation
- [x] Build database
- [x] Export Center
- [x] Database Health
- [x] Icon rendering

## v0.2 — Data Quality and Navigation
- [ ] Search improvements
  - Improve how users find personas, speeds, modifiers, reference IDs, and pricing sets from a single search box.
- [ ] Filter builder
  - Let users combine filters such as family group, pricing set, speed tier, modifier, equipment, and symmetrical speed.
- [ ] Duplicate finder
  - Identify duplicate or near-duplicate persona, speed, pricing, disclaimer, and icon records before they affect exports.
- [ ] Pricing validator
  - Validate pricing schedules for missing rows, duplicate rows, overlapping month ranges, mixed promotion data, and unexpected FREE pricing.

## v0.3 — Flyer Production
- [ ] Flyer designer
  - Provide a design workspace for assembling persona-specific flyers from offers, icons, modifiers, disclaimers, and market content.
- [ ] PDF export
  - Generate consistent PDF output with print-safe layout, page-break handling, and predictable file naming.
- [ ] Template manager
  - Save, reuse, preview, and version flyer templates by pricing set, family group, or campaign.

## Current Priority

**Search improvements** are the next recommended task because they will make the existing persona database easier to navigate before adding more advanced filtering, duplicate detection, or flyer-production tools.
